Smart Cache and full security your site
== Description ==

Smart Cache and full security your site. Some plugin cache not work fine with security plugin, so We Created this plugin. It work fine on Cache, Security and mobile plugin

Supported features:

* Cache
<ul>
<li>Cache on server</li>
<li>Cache on browser</li>
<li>Store compressed pages</li>
<li>Send compressed pages</li>
</ul>


* Security
<ul>
<li>Smart ban IP locked by other site install this plugin(sync ban ip)</li>
<li>Ban Ip, User Agents</li>
<li>Cache on browser</li>
<li>Limit Login</li>
<li>Notification if login limited</li>
<li>Enforce Front end, Back end, Login with SSL</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the  menu in WordPress
